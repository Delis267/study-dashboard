# Studien-Dashboard - Phase 1

Dieses Repository enthaelt Arbeitsmaterialien fuer die Konzeptionsphase eines prototypischen Studien-Dashboards.
## Ziel

Das Dashboard soll spaeter helfen, den Studienfortschritt im Blick zu behalten:

- Abschlussziel innerhalb von 36 Monaten
- Ziel-Notendurchschnitt 2,0
- ECTS-Fortschritt
- bisherige Studiendauer ab Startdatum
- Velocity in ECTS pro Monat
- Prognose der offenen Monate
- Moduluebersicht mit Status und Note

## Inhalte

- `src/csv_repository.py`: kleiner CSV-Machbarkeitstest
- `src/prognose_test.py`: einfacher Test fuer Kennzahlen und Prognose
- `src/gui_test.py`: einfacher Tkinter-Test fuer die grafische Darstellung

## Rahmenbedingungen

- Python 3.10+
- nur Python-Standardbibliothek
- CSV als einfache Datenspeicherung
- Tkinter fuer den GUI-Machbarkeitstest
